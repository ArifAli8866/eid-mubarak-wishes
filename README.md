## 🌙 What is this?

<img align="right" width="340" src="https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExcDd5dGJ6enZ4Z2Z4YWxsNW9qOHJ5NHp5aGkzMWd5YmZob3B6NHoxbiZlcD12MV9pbnRlcm5hbGdfaWZfaWQ&ct=g/l0MYt5jl4bekpVPNC/giphy.gif" />

**Eid Mubarak Wishes** is a beautiful, fully client-side web application that lets you create deeply **personalised Eid greeting cards** for your loved ones — no login, no backend, no API key, no cost.

You answer a few heartfelt questions about the person you are wishing, and the app crafts a **unique, personal message** just for them — then lets you download it as a PNG card or share it via a magic link 🔗

> 💡 Built as a **single HTML file** — works offline, deploys anywhere, zero dependencies.

## 🚀 How It Works

<div align="center">

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                   │
│   Step 1 🌙          Step 2 💬          Step 3 ⌨️               │
│  ─────────────      ────────────────    ──────────────────       │
│  Pick the           Answer personal     Watch the wish           │
│  relation           questions about     type out live            │
│  (14 options)       your loved one      on screen                │
│                                                                   │
│   Step 4 🌸          Step 5 🔗                                   │
│  ─────────────      ────────────────                             │
│  Edit • Download    Share a magic                                │
│  your card as PNG   link with anyone                             │
│                                                                   │
└─────────────────────────────────────────────────────────────────┘
```

</div>

### 1️⃣ Pick the Relation

Choose from **14 beautifully themed relation cards** — each one has its own unique pastel colour palette.

### 2️⃣ Answer Personal Questions

Each relation has a **custom set of questions** designed to capture what makes that relationship unique:

- For **Mother** → Her name, her most special quality, something she does you cherish, your prayer for her
- For **Friend** → Their name, how long you've been friends, your friendship vibe, something only you two share
- For **Spouse** → Their name, how long you've been together, what you love most, your deepest wish for them

### 3️⃣ Watch the Magic ✨

The app generates a **personalised wish** using your answers and plays it out with a beautiful typewriter animation — letter by letter, as if someone is writing it just for you.

### 4️⃣ Download or Share

- **Download as PNG** — a high-quality image card ready to send on WhatsApp, Instagram, or anywhere
- **Get a share link** — anyone who clicks the link sees the full beautiful card in their browser

## 📦 Deploy in 3 Minutes

### Option 1 — Vercel (Recommended) ⚡

```bash
# 1. Fork or clone this repo
git clone https://github.com/YOUR_USERNAME/eid-mubarak-wishes.git

# 2. Push to GitHub (if not already)
git add .
git commit -m "🌙 Initial commit"
git push origin main

# 3. Go to vercel.com → New Project → Import your repo → Deploy
# That's it! Your site is live 🎉
```

### Option 2 — GitHub Pages 📄

```bash
# In your repo settings:
# Settings → Pages → Source → Deploy from branch → main → / (root) → Save
# Your site will be at: https://YOUR_USERNAME.github.io/eid-mubarak-wishes/
```

### Option 3 — Run Locally 💻

```bash
# Just open the file — no server needed!
open eid-wishes.html

# Or with a simple server
npx serve .
# → http://localhost:3000/eid-wishes.html
```

## 🔗 How Sharing Works

The share link stores everything in the **URL itself** as query parameters:

```
https://your-site.vercel.app/eid-wishes.html?rel=mother&name=Ammi&wish=Your+wish+text
```

- ✅ No database needed
- ✅ No server needed  
- ✅ Works on any static host
- ✅ Link works forever as long as the site is live
- ✅ Receiver sees the full beautiful card when they open it
